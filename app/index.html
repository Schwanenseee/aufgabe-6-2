Was ich bisher weiss über Cloud

Eine Cloud ist einfach eine Rechenressource, welche nicht bei mir lokal ist auf dem Rechner, sondern auf irgendeiner Form externer Server.

Das kann z.B. ein ESXi-Host sein, wie beim BZU, oder das kann bei Azure gehostet sein oder oder oder...

Es gibt grundsätzlich 4 verschiedene Modelle von Cloud:

IaaS: Infrastructure as a Service: jemand stellt mir eine Infrastruktur zur Verfügung, z.B. über ILO, und ich kann dann darauf zugreifen. ich bin dann nicht verantwortlich für z.B. Gebäudesicherheit etc.

PaaS: Platform aaS: jemand stellt mir z.B. eine Windows VM zur Verfügung, und darauf habe ich dann Admin Rechte. Ich bin dann auch nicht verantwortlich für Installation OS etc.

SaaS: Software aaS: jemand stellt mir eine Software zur Verfügung, z.B. eine Moodle Instanz, wo ich Admin Rechte habe. ich bin dann nicht verantwortlich für OS Updates etc.

FaaS: Function aaS: jemand stellt nur eine einzelne Funktion zur Verfügung, z.B. für Entwickler
